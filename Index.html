Navegadores Web

Un navegador web (en inglés, web browser), es un software, aplicación o rograma que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari: Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS

Mozilla Firefox es un navegador web libre y de codigo abierto desarrollado para Linux, Android, iOS, macOS y Microsoft Windows coordinador por la Corporación Mozilla y la Fundación Mozilla

Google Chrome Google Chrome es un navegador web de código abierto desarrollado por Google, aunque dervado de proyectos de código abierto
